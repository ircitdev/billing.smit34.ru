# СмИТ Биллинг 1.0 — Презентация для клиентов

## Рекомендуемый формат
- **16:9** (1920x1080)
- **Шрифт заголовков**: Inter Bold / Montserrat Bold
- **Шрифт текста**: Inter Regular / Montserrat Regular
- **Стиль**: светлый, чистый, корпоративный минимализм

### Цветовая палитра (светлая тема)

| Элемент | Цвет | HEX |
|---------|------|-----|
| Фон слайда | Белый | `#FFFFFF` |
| Фон альтернативный | Светло-серый | `#F8FAFC` |
| Заголовки | Тёмно-синий | `#0F172A` |
| Основной текст | Тёмно-серый | `#334155` |
| Второстепенный текст | Серый | `#64748B` |
| Акцент основной | Emerald | `#10B981` |
| Акцент дополнительный | Teal | `#14B8A6` |
| Акцент градиент | Emerald → Teal | `#10B981 → #14B8A6` |
| Карточки | Белый с тенью | `#FFFFFF`, shadow `0 4px 20px rgba(0,0,0,0.06)` |
| Разделители | Светло-серый | `#E2E8F0` |
| Бейджи/метки | Emerald 8% фон | `rgba(16,185,129,0.08)`, текст `#059669` |
| Ошибка/минус | Красный | `#EF4444` |
| Успех/плюс | Emerald | `#10B981` |

### Оформление слайдов
- Фон: белый `#FFFFFF` или чередование с `#F8FAFC`
- Заголовки: `#0F172A`, 36–44px, Bold
- Текст: `#334155`, 18–22px, Regular
- Акцентные элементы: emerald `#10B981` (линии, иконки, подчёркивания)
- Карточки: белые с лёгкой тенью и скруглёнными углами (16px)
- Иконки: линейные (stroke), цвет emerald или тёмно-серый
- Нумерация слайдов: внизу справа, `#94A3B8`, 12px
