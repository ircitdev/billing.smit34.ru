Возможности

Всё для управления ISP

Полный цикл: от подключения абонента до СОРМ-отчётности. Один интерфейс, один сервер, без лишнего.

👤

Управление абонентами

Древовидная структура, папки, категории. Поиск, фильтры, быстрые чипы. Массовые операции, экспорт в CSV.

💰

Тарификация и платежи

Гибкие тарифные планы, периодические и разовые услуги. Интеграция с платёжными системами, 1С, SOAP API.

🔌

RADIUS-авторизация

FreeRADIUS 3.2.3 из коробки. PPPoE, IPoE, VoIP, IPTV. Три виртуальных сервера для разных типов трафика.

📈

Аналитика и отчёты

Dashboard с ключевыми метриками. Должники, баланс по категориям, топ тарифов. Экспорт данных в Excel.

🔒

СОРМ-интеграция

Передача данных по ФЗ. Поддержка стандартных и пользовательских схем СОРМ3. Готовые коннекторы.

📧

Уведомления

SMS и email рассылки. Автоматические напоминания о задолженности. Шаблоны сообщений с поддержкой переменных.
